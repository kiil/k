Merging an emoji üòò font into a text font ‚ù§Ô∏è

23 May 2023·97 words·1 min

I wanted to use the font Noto Sans with the Noto Emoji font for making an image overlay text in Hugo.

I used Fontforge to merge the two fonts.

At first the emoji font was much larger than the text font.

In my case it only worked properly when

  1. Opening the text font first with ‘File -> Open File’.
  2. Then for ‘Element -> Font Info’ under ‘General’ setting ‘Em Size’ to 2048.
  3. Now using ‘Element -> Merge Fonts’ on the emoji font to merge the two.
  4. Finally creating the merged font file with ‘File -> Generate Fonts’.
